## `OllamaProvider` 구현

### 세션 주입 설계
```python
class OllamaProvider:
    def __init__(
        self,
        base_url: str = "http://localhost:11434",
        session: aiohttp.ClientSession | None = None,
    ) -> None:
        self._owns_session = session is None
```
- `session=None` (기본): `async with` 진입 시 세션 자동 생성, `__aexit__` 시 자동 닫기
- `session=<mock>` (테스트): 외부 세션 직접 주입 → `async with` 없이도 사용 가능

이 설계 덕분에 `aioresponses` 등 추가 의존성 없이 `unittest.mock`만으로 HTTP를 완전히 모킹할 수 있음.

### health_check
`/api/tags` 조회 → models 목록 비어있으면 False → 예외 발생해도 False (앱 시작 시 gate용).

### mypy 이슈 해결
`kwargs.get("temperature", 0.3)` → 반환 타입이 `object` → `float(object)` mypy 오류.
`isinstance` 체크 후 조건부 변환으로 해결:
```python
raw_temp = kwargs.get("temperature", 0.3)
temperature = raw_temp if isinstance(raw_temp, float) else float(str(raw_temp))
```
