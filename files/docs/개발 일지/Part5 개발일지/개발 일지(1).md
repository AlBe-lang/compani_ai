## `SQLiteStorage` 구현

### 마이그레이션 시스템
```
adapters/migrations/
└── 001_initial_schema.sql   ← kv_store 테이블 정의
```
- `_migrations` 테이블로 적용 이력 관리
- 파일명 정렬 순서대로 실행, 중복 적용 방지

### StoragePort 구현
| 메서드 | 동작 |
|---|---|
| `save(key, value)` | INSERT OR REPLACE — 중복 키 덮어쓰기 |
| `load(key)` | SELECT, 없으면 None 반환 |
| `update(key, value)` | UPDATE (기존 키 갱신) |
| `query(**filters)` | 전체 로드 후 Python-side 필터링 |

값은 모두 JSON 직렬화 (`json.dumps(default=str)`)로 저장.

## `InProcessEventBus` 구현

```python
class InProcessEventBus:
    def subscribe(self, event_type: str, handler: EventHandler) -> None: ...
    async def publish(self, event_type: str, payload: EventPayload) -> None: ...
```

- `defaultdict(list)`로 이벤트 타입별 핸들러 목록 관리
- 동기 핸들러, 비동기 핸들러 모두 지원 (`asyncio.iscoroutine` 체크)
- 핸들러 예외는 로그만 남기고 계속 진행 (다른 구독자 차단 방지)
