### Stage 1 검증 조건
| 검증 항목 | 매칭 키워드 |
|---|---|
| Dockerfile 존재 | path == "Dockerfile" |
| FROM 인스트럭션 | `from ` (lower) |
| WORKDIR 인스트럭션 | `workdir` (lower) |
| 비루트 유저 | `user ` (lower) |
| pip no-cache-dir | `--no-cache-dir` |
| .dockerignore 존재 | path == ".dockerignore" |

### `_normalize_payload` 오버라이드
`env_vars_required`와 `ports_exposed`는 `TaskResult(extra="forbid")` 통과를 위해 검증 후 제거:
```python
def _normalize_payload(self, task, payload):
    normalized = super()._normalize_payload(task, payload)
    normalized.pop("env_vars_required", None)
    normalized.pop("ports_exposed", None)
    return normalized
```
