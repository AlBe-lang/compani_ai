## 3. 결정 이유

- **Stage 6 트랙 분리**: 테스트 생성 단계는 React/Flutter 구현과 독립적. 누적 검증 강제 시 테스트 파일만 포함한 응답이 구현 파일 미포함으로 실패하는 구조적 문제
- **`testWidgets()` 강제**: Flutter 위젯 테스트의 핵심 API. 단순 `test()` 만으로는 위젯 렌더링 없이도 통과 가능하므로 추가 조건
- **`@testing-library/react` 확인 (`render` or `renderHook`)**: import 문 외에 실제 사용 여부 확인
- **전략 변경 없음**
