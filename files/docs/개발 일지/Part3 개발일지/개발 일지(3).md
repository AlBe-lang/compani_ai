## 3. 결정 이유

- **`api_endpoints_used` 필수 검증**: 규칙 문서 3.5절에 명시. Part 5 통합 시 Backend ↔ Frontend 계약 일치 자동 검증에 사용 예정

- **로딩/에러 상태는 컴포넌트 파일 기준**: API 모듈 자체가 아닌 UI 컴포넌트에서 상태를 관리해야 함
- **전략 변경 없음**
