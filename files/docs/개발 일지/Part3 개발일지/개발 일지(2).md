## 3. 결정 이유

- **Custom hook `/hooks/` 경로 강제**: 컴포넌트 파일에 useState가 있는 경우도 검증 통과시키되, custom hook은 반드시 별도 파일(`/hooks/*.ts`)로 분리해야 함 — 아키텍처 분리 원칙
- **`useState` 키워드 전체 파일 검색**: `useState`는 hooks 파일 또는 컴포넌트 어디에나 올 수 있으므로 경로 필터 없이 전체 검색. 특정 경로 제한보다 유연함
- **전략 변경 없음**
