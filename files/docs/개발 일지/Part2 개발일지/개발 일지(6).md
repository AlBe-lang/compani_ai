## 2. 기술적 결정 사항

### A. Unit 테스트 경로 제한 (`/tests/unit/`)
**이유**: integration test 파일도 `def test_*` 함수를 포함하므로, `has_unit_tests`를 단순히 `def test_` 존재 여부로만 체크하면 unit test 파일 없이도 통과됨.  
`/tests/unit/` 하위 경로 파일로 범위를 좁혀 두 종류를 구분.

### B. Integration 테스트는 경로 무관, 내용 기준
httpx/AsyncClient/TestClient는 integration test 전용 라이브러리로 어느 경로에 있어도 유효.  
경로보다 내용 패턴이 더 신뢰할 수 있는 기준.
